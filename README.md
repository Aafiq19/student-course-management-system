# 🎓 Student Course Management System (SCMS)

A full-stack Student Course Management System built using **React**, **Spring Boot**, **JWT Authentication**, and **PostgreSQL**.

This project is developed as a university learning project following production-inspired architecture and Git collaboration practices.

---
## 🚀 Tech Stack

### Frontend

- React JS
- React Router
- Axios
- ShadCN UI
- React Hook Form

### Backend

- Spring Boot
- Spring Security
- JWT Authentication
- Spring Data JPA

### Database

- PostgreSQL

- Add Student

- Update Student
- Delete Student

### Course Management

- Add Course

- Update Course
- Delete Course

### Enrollment Management

- Enroll Student into Course
- Remove Enrollment
- View Enrollments

- Total Students
- Total Courses
- Total Enrollments
